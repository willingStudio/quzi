# 取字

Chrome扩展程序 - 取字-快速识别,一秒取字

快速准确识别图片上的文字,方便文字复制编辑

您可以利用这款插件在线把图片或者视频中的文字进行截取并自动识别，数秒之内即转换为可编辑操作的文本，提高工作效率，目前这款插件支持识别中文和英文两种语言，并可兼容所有版本的Chrome浏览器。
